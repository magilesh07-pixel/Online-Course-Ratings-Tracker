# Course Ratings Hub | Intelligence Dashboard

A modern, sleek web application built with **Django** that allows users to track and analyze course ratings. This project provides a clean interface for managing course feedback and visualizing average performance.

## 🚀 Features

- **Course Management**: Add and delete courses with ease.
- **Rating System**: Rate courses on a scale of 1 to 5.
- **Real-time Analytics**: Automatically calculates the average rating of all courses.
- **Sleek UI**: A premium, responsive design with modern aesthetics.
- **Report Generation**: A dedicated PDF-ready view for course reports.

## 🛠️ Technology Stack

- **Backend**: Django (Python)
- **Frontend**: HTML5, CSS3 (Vanilla)
- **Database**: SQLite (Default)

## 📋 Prerequisites

- Python 3.x
- Django

## 🔧 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Course-Ratings-Hub.git
   cd Course-Ratings-Hub
   ```

2. **Install dependencies**:
   ```bash
   pip install django
   ```

3. **Run Migrations**:
   ```bash
   python manage.py migrate
   ```

4. **Start the server**:
   ```bash
   python manage.py runserver
   ```

5. **Access the app**:
   Open `http://127.0.0.1:8000` in your browser.
